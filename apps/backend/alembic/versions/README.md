# Alembic Migrations

This directory contains database migration files generated by Alembic.

## Creating a New Migration

```bash
# Auto-generate migration from model changes
poetry run alembic revision --autogenerate -m "description of changes"

# Create empty migration
poetry run alembic revision -m "description of changes"
```

## Running Migrations

```bash
# Upgrade to latest
poetry run alembic upgrade head

# Upgrade by one version
poetry run alembic upgrade +1

# Downgrade by one version
poetry run alembic downgrade -1

# Show current version
poetry run alembic current

# Show migration history
poetry run alembic history
```

## Migration Naming Convention

Migrations are automatically named with format:
`YYYY_MM_DD_HHMM-{revision_id}_{slug}`

Example: `2026_06_03_1430-abc123def456_initial_migration`
